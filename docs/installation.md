# Installation

## From PyPI

```bash
pip install logsuite
```

## From Source

```bash
git clone https://github.com/kkollsga/logsuite.git
cd logsuite
pip install -e .
```

## Requirements

- Python 3.10+
- numpy >= 1.20
- pandas >= 1.3
- scipy >= 1.7
- matplotlib >= 3.5

## Development Installation

```bash
pip install -e ".[dev]"
```

## Documentation Build

```bash
pip install -e ".[docs]"
cd docs
make html
```
